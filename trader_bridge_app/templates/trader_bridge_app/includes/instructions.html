The Stock
  • The stock pays a dividend at the end of each period. Dividends are equally likely to be {{ dividend_values_text }}.
  • Each period, the dividend on each stock is {{ expected_dividend }} on average. Since there are {{ num_days }} periods, the sum of all expected dividends in period 1 is {{ fundamental_value_start }} (= {{ num_days }} * {{ expected_dividend }}) for each stock. {% if num_days_after_first > 0 %} In the second period the stock will have paid its first dividend, so there are only {{ num_days_after_first }} more payments left, each worth {{ expected_dividend }} in expectation. So, the sum of all remaining expected dividends is {{ fundamental_value_after_first }}. {% endif %} {% if num_days_after_second > 0 %} In the third period there will be {{ num_days_after_second }} more payments left, with an expected total dividend of {{ fundamental_value_after_second }}, etc. {% endif %} The fundamental value declines over time and equals expected dividend times remaining periods. After the final dividend is paid, the stock is worthless.
Trading Rules

Trading uses a limit order book. You may submit buy/sell limit orders or trade immediately against resting orders.

  • You may buy shares only if you have sufficient cash.
  • You may sell shares only if you own them (no short selling).
  • You cannot borrow additional cash.
Price Forecasting

On forecast daybreaks, after dividends are paid, you submit a point forecast for the next-period end price and report confidence. If your forecast is within {{ forecast_bonus_threshold_pct }}% of the realized price, you earn a bonus of E${{ forecast_bonus_amount }}. {{ forecast_schedule_text }}

Payment
  • Final payoff equals your cash holdings at the end of a randomly selected market, including period {{ payoff_period }} dividends, plus forecast bonuses earned across all markets.
  • Experimental currency converts to real money at rate: 1 point = {{ exchange_rate_text }} (session currency units).
  • Quiz bonus per correct answer: {{ quiz_bonus_per_correct_text }} points.
Important Notes
  • Your trading screen shows order book, recent transactions, cash balance, and share holdings.
  • Each period lasts exactly {{ trading_day_duration }} minute{% if trading_day_duration != 1 %}s{% endif %}. A timer shows time remaining.
  • Markets are independent. Each new market starts with a fresh endowment and the stock fundamental value resets.
